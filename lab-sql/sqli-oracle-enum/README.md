# SQL Injection Oracle Enumeration Lab

## Deskripsi
Lab ini menguji kemampuan peserta dalam melakukan enumerasi struktur database Oracle melalui teknik SQL Injection. Peserta harus menggunakan teknik enumerasi Oracle untuk menemukan tabel dan kolom yang berisi informasi sensitif.

## Tujuan
- Mengidentifikasi bahwa aplikasi menggunakan database Oracle
- Menggunakan teknik enumerasi Oracle untuk mengeksplorasi struktur database
- Menemukan tabel dan kolom yang berisi informasi sensitif
- Mendapatkan flag melalui enumerasi Oracle

## Teknologi
- Python Flask
- Oracle Database
- Docker

## Cara Menjalankan
1. Pastikan Docker dan Docker Compose terinstal
2. Jalankan perintah: `docker-compose up -d`
3. Akses aplikasi di http://localhost:5007
4. Setelah menyelesaikan tantangan, flag akan ditampilkan

## Konfigurasi CTF
- Port: 5007
- Flag: `IDS{r1s2t3u4v5w6x7y8z9a0b1c2d3e4f5g}`
- Kategori: Web Exploitation
- Tingkat Kesulitan: Sulit
